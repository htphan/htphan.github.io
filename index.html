PORTFOLIO

Convoy
Code
Convoy

Have you ever had a hard time trying to coordinate picking up your kids? Convoy makes it easier! Convoy is a web and mobile application that assists in the coordination of carpools for all ages. It provides users with the ability to input pertinent information like medical information and emergency contacts. Other features include map display, distance filters, and a social bulletin board for every carpool.

The server-side architecture was built with Rails to provide a RESTful API that was consumed by the Front-End and Mobile applications.

Tech: Ruby, Rails, PostgreSQL, Mandrill, Redis, Resque, Resque-Scheduler, Mailer, JBuilder, Heroku

DCRYPTr
Code Demo
DCRYPTr

DCRYPTr is a guessing game where a user is able to upload some abstract image through their mobile device and upload it to our server with an answer that they equate to the image. A player is then able to use our web platform to access a library of these images and has to guess them correctly. Each progressive incorrect answer nets the player fewer points, but they cannot lose point they have already earned!

The server-side architecture was built with Rails to provide a RESTful API that was consumed by the Front-End and Mobile applications.

Tech: Ruby, Rails, PostgreSQL, Amazon AWS S3, JBuilder, Heroku
